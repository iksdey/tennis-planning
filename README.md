# tennis-planning

Outil permettant d'estimer la plage de participation à un tournoi de tennis.

Le calcul s'appuie sur les informations suivantes :
- date de début du tournoi ;
- date de fin du tournoi ;
- classement minimum autorisé ;
- classement maximum autorisé ;
- classement du joueur.

L'objectif est d'estimer la période pendant laquelle le joueur est susceptible de participer au tournoi.

## Structure du projet

- `tennis_planning.ipynb` : notebook principal.
- `planning.py` : fonctions Python utilisées par le notebook.

## Prérequis

- Python 3.x
- JupyterLab
